A navigation bar for Emacs, inspired by [[http://getbootstrap.com/][Bootstrap's]] navbar component.

* Overview

navbar.el provides a customizable navigation bar that sits at the top of your Emacs frame. Like Bootstrap's navbar, it supports various components called "navbar items" that can display dynamic content such as time, buffer information, or workspace tabs.

* Requirements

- Emacs 24.4 or later

* Installation

** From MELPA

#+begin_src emacs-lisp
(use-package navbar
  :ensure t
  :config
  (navbar-mode 1))
#+end_src

** Manual Installation

Clone this repository and add it to your load path:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/navbar.el")
(require 'navbar)
(navbar-mode 1)
#+end_src

* Usage

** Basic Configuration

Define navbar items and enable the mode:

#+begin_src emacs-lisp
(setq navbar-item-list '("Hello, world!"))
(navbar-mode 1)
#+end_src

** Dynamic Items

Navbar items can be functions that return property lists. Use =navbar-sync= to refresh the display after modifying =navbar-item-list=.

The =navbarx-time= item displays the current time in the navbar instead of the mode line:

#+begin_src emacs-lisp
(require 'navbarx-time)
(setq navbar-item-list '(navbarx-time))
(navbar-mode 1)
#+end_src

The =navbarx-elscreen= item displays ElScreen tabs in the navbar instead of the header line:

#+begin_src emacs-lisp
(require 'navbarx-elscreen)
(setq navbar-item-list '(navbarx-elscreen))
(navbar-mode 1)
#+end_src

** Customization

Use =M-x customize-group RET navbar RET= to explore available customization options.
